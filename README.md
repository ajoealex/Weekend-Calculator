# Weekend Countdown

A live countdown timer showing how much time is left until the weekend (Saturday).

## Features

- Live countdown displaying days, hours, minutes, and seconds until Saturday
- Copy button to share the countdown text
- Animated progress bar showing week progress (Monday to Saturday)
- Weekend celebration message when it's Saturday or Sunday
- Dark theme with animated gradient background

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Build Output

The production build outputs to the `docs/` folder with relative asset paths (`./`), making it ready for GitHub Pages or similar static hosting.
